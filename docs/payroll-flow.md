# Payroll Flow

## Step-by-Step Execution

### 1. Input Data
- Employee details
- Salary structure
- Attendance
- Performance rating

---

### 2. Salary Rule Engine
- Basic salary
- HRA calculation
- Allowances
- Performance bonus

---

### 3. Gross Salary Calculation
- Add all components
- Deduct LOP
- Add overtime

---

### 4. Deduction Engine
- PF (12%)
- ESI
- Professional Tax

---

### 5. Tax Engine
- Annual income calculation
- Tax slab application
- Monthly TDS

---

### 6. Arrears Processing
- Salary revisions
- Backdated adjustments

---

### 7. Final Calculation

Net Salary =  
Gross Salary - Deductions - Tax + Arrears

---

### 8. Output
- Payslip object generated
