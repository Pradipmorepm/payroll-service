#  Payroll Service

##  Overview
This service handles core payroll calculation logic including:
- Salary computation
- Deductions
- Tax calculation
- Bonus & arrears

---

##  Architecture
- Engine Layer → Calculation rules
- Service Layer → Flow orchestration
- Model Layer → Domain objects

---

##  Features
- Modular payroll engine
- Configurable rules
- Unit tested
- Scalable design

---

mvn test
